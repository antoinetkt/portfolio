Le projet "REST API Pizzaland" consiste à créer une API REST pour un service de livraison de pizza. L'API doit permettre de gérer les commandes, les pizzas et les ingrédients du pizzeria. Le projet "Doctolibre" consiste à créer une application web de gestion de rendez-vous médicaux.
